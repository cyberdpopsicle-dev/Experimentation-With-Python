# 🌟 Stellar Lifeline Generator

An interactive astrophysics simulation tool that dynamically models and visualizes a star's entire lifecycle—from protostar formation to its ultimate cosmic remnant—based on user-defined physical properties.

## 🚀 Features

* **Dynamic Physics Engine:** Inputs parameters like Mass ($M_\odot$), Metallicity ($Z$), and Binary status using real scaling laws ($L \propto M^{3.5}$).
* **Multi-Panel Visual Dashboard:** Generates three coordinated plots in one window:
  * **Graphical Timeline:** A visual "swimlane" showing changes in stellar size, color, and milestone stages.
  * **Core Fusion Heatmap:** A Seaborn heatmap tracking element abundance and nuclear fusion transitions over time.
  * **H-R Diagram Track:** A logarithmic Hertzsprung-Russell track plotting the star's precise temperature vs. luminosity trajectory.

## 📦 Prerequisites

Ensure you have Python installed along with the following libraries:

```bash
pip install matplotlib seaborn numpy

```

## 🛠️ Usage

1. Clone this repository and navigate to the directory:
```bash
git clone [https://github.com/yourusername/stellar-lifeline-generator.git](https://github.com/yourusername/stellar-lifeline-generator.git)
cd stellar-lifeline-generator

```


2. Run the main script:
```bash
python stellar_generator.py

```


3. Enter your parameters when prompted:
* **Mass:** $0.08$ to $150 \ M_\odot$
* **Metallicity:** $0.01$ to $3.0$
* **Binary Companion:** `yes` or `no`



## 📊 Sample Output

The script outputs a unified dashboard displaying the evolutionary sequence of the star, its changing core chemistry, and its path across the H-R diagram based on your exact specifications.
