# 🌟 Stellar Lifeline Generator

An interactive astrophysics simulator that models and visualizes a star's complete lifecycle based on user-defined properties.

## 🚀 Features

* Dynamic stellar evolution using mass, metallicity, and binary parameters.
* Visual lifecycle timeline with key evolutionary stages.
* Core fusion heatmap showing elemental changes.
* Hertzsprung–Russell (H-R) diagram track of temperature and luminosity.

## 📦 Installation

```bash
pip install matplotlib seaborn numpy

## 🛠️ Usage

```bash
git clone https://github.com/cyberdpopsicle-dev/stellar-lifeline-generator.git
cd stellar-lifeline-generator
python stellar_generator.py
```

Enter:

* Mass: `0.08–150 M☉`
* Metallicity: `0.01–3.0`
* Binary Companion: `yes/no`

## 📊 Output

A unified dashboard displaying the star's evolutionary timeline, fusion history, and H-R diagram track.
