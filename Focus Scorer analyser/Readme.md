# Focus Score Analyzer

A Python-based productivity analytics tool that simulates user activity and computes a **Focus Score** using keystroke frequency patterns.

### Key Highlights

* Object-Oriented Design (OOP)
* Data Simulation & Analysis
* Statistical Scoring Mechanism
* Performance Classification Logic
* Extensible for Real-Time Monitoring and ML Integration

### Interview Relevance

This project demonstrates:

* Data modeling and processing
* Algorithmic decision-making
* Software design principles
* Practical application of statistics in analytics systems

### Future Scope

* Real-time activity tracking
* Machine Learning-based productivity prediction
* Interactive dashboards (Streamlit/Flask)
* Behavioral analytics and recommendations
